# 🌾 AI Smart Agriculture App

## 📌 Overview

This project is an AI-powered Crop Recommendation System that suggests the most suitable crop based on soil nutrients and weather conditions. It also provides profit estimation, explanation, and farmer assistance through a chatbot.

---

## 🚀 Features

* 🌾 Crop Prediction using Machine Learning (Random Forest)
* 💰 Profit Estimation per acre
* 📊 Confidence Score of prediction
* 🤖 Explainable AI (reason behind recommendation)
* 📈 Feature Importance Visualization
* 🖼️ Crop Image Display
* 💬 Farmer Assistant Chatbot

---

## ▶️ How to Run Locally

1. Clone the repository
   git clone https://github.com/sanjaykurakul/ai-smart-agriculture-app.git

2. Navigate to project folder
   cd ai-smart-agriculture-app

3. Install dependencies
   pip install -r requirements.txt

4. Run the app
   streamlit run app.py

## 🧪 Model Details

Algorithm: Random Forest Classifier

Input Features:

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH Level
* Rainfall
